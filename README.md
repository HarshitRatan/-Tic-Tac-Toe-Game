# Tic-Tac-Toe Game
Code by Harshit Ratan Shukla.

Tic Tac Toe is a two-player game in which the objective is to take turns and mark the correct spaces in a 3x3 grid.
This Game has two option to play :-
i) Against human
ii) Against Computer.
